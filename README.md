# 投放素材批量生成工具
适用于巨量、小红书、百度等平台的广告素材快速批量制作工具，无需安装Python，打开即用。

## 核心功能
✅ 多分类素材自由拖拽组合
✅ 批量导出时强制图片两两差异达标
✅ 自定义画布尺寸（适配全平台）
✅ Mac 直接运行，绿色无安装
✅ 一键单张导出 / 批量导出

## 使用方法
1. 下载页面右侧 Releases 里的软件包
2. 在软件同目录新建 `materials` 文件夹
3. 在 materials 里放入你的素材分类文件夹（比如「背景」「文案」「元素」）
4. 双击打开软件即可使用
